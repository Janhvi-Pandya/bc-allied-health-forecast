Methods & Technical Details

Analytical Framework

This page documents the methodological choices made in this analysis and the reasoning behind them.

Data Quality Strategy

Why Group-Median Imputation?

Method Pros Cons Used here?
Group-median Robust to outliers; preserves local distribution Doesn’t capture trend Yes
Group-mean Easy to compute Sensitive to outliers (e.g., one anomalous recruitment year) No
Global median Simple Ignores real HA-level variation (Northern ≠ Fraser) No
Multiple imputation Statistically principled Overkill for ~3% missingness with no systematic pattern No
Deletion No assumptions Loses data; can bias results if not MCAR No

The missing rate (~3%) is low and shows no systematic pattern by group, making group-median a defensible and practical choice.

Regression Modeling

Specification

\[ \text{headcount}_{\text{profession}} = \beta_0 + \beta_1 \cdot \text{year} + \varepsilon \]

Fitted separately per profession on provincial aggregates (summed across all Health Authorities).

Why Exclude COVID Years?

The 2020–2021 observations reflect a temporary shock (pandemic disruption), not the underlying structural trend. Including them would:

  • Reduce the estimated growth rate (\(\hat{\beta}_1\))
  • Widen prediction intervals unnecessarily
  • Bias forecasts downward

Instead, COVID’s impact is quantified separately via Welch’s t-tests on vacancy rates — a cleaner analytical separation.

Why Linear Regression?

For 13 training observations (15 years minus 2 COVID years) with steady growth trends, linear regression is appropriate and interpretable. More complex models (polynomial, ARIMA) risk overfitting on this sample size and would add complexity without clear benefit.

Prediction Intervals vs Confidence Intervals

This analysis uses prediction intervals (not confidence intervals) for forecasts:

  • Confidence interval: uncertainty about the mean response — “where is the regression line?”
  • Prediction interval: uncertainty about a new observation — “where will next year’s headcount actually land?”

Prediction intervals are wider because they account for both model uncertainty and individual-observation variability. For workforce planning, prediction intervals are the correct choice: planners need to know the realistic range of outcomes, not just where the average sits.

Statistical Testing

Welch’s t-test Design

Parameter Value Rationale
Test Welch’s t-test Does not assume equal variances between groups
Alternative One-sided (greater) Directional hypothesis: COVID increased vacancy rates
Pre-COVID window 2015–2019 5 years of stable pre-pandemic data
Post-COVID window 2022–2024 Excludes 2020–2021 (acute disruption, not steady state)
α 0.05 Standard significance level

The one-sided test is justified by the clear directional prior: there is no plausible mechanism by which a pandemic would decrease vacancy rates.

Reproducibility

All random processes use set.seed(42) (data generation) and set.seed(99) (missing value injection). Running the pipeline on any machine with the same R version and packages will produce identical results.

Rscript 00_generate_data.R   # Always produces the same CSV
Rscript 01_analysis.R        # Always produces the same plots and tables
